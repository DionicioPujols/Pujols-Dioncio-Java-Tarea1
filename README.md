## 📝 Descripción
Esta es una aplicación de consola desarrollada en Java que contiene un menú interactivo con múltiples opciones. Cada opción del menú ejecuta uno de los ejercicios prácticos asignados correspondientes a los capítulos 1 al 5 del libro *Java 2 Manual de Programación*. La aplicación está diseñada bajo el paradigma de Programación Orientada a Objetos y permite al usuario navegar de forma continua entre las diferentes funcionalidades hasta que se selecciona la opción de salida.

## 🛠️ Tecnologías utilizadas
- ☕ Java
- 📦 JDK 17
- 💻 Visual Studio Code

## 📋 Opciones del Menú
1. Calculadora básica
2. Número par o impar
3. Tabla de multiplicar
4. Contador de vocales
5. Promedio de notas
6. Clase Estudiante
7. Cuenta bancaria
8. Herencia de vehículos
9. Matriz 3x3
10. Sistema básico de inventario
11. Salir de la aplicación
